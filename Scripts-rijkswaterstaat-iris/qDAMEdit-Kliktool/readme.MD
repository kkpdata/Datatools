# Doel

Karakteristieke punten klikken in dwarsprofielen.


# Gebruik

De Kliktool kan worden gebruikt om karakteristieke punten te klikken op dwarsprofielen. De tool kan zowel voor DAM als voor WBI worden gebruikt. Voor WBI kan de tool worden ingezet om invoer te genereren voor het maken van pipingberekeningen. Riskeer vraagt voor berekeningen voor het toetsspoor piping (STPH) om profielschematisaties; een bestand met de hoogtegegevens van het dijkprofiel en een karakteristieke puntenbestand voor berekeningen voor het toetsspoor piping (STPH). Met de Kliktool kan dit worden gegenereerd. 


# Functionaliteit

Het invoer bestand is een csv-bestand met per regel alle punten van het gehele dwarsprofiel. In de tool worden de dwarsprofielen getoond, waarna er eenvoudig karakteristieke punten kunnen worden geklikt. Uiteindelijk kunnen de volgende bestanden worden opgeslagen: surfacelines.csv, characteristicpoints.csv, een logbestand.csv en de database.ded. Het is tevens mogelijk om plaatjes van de dwarsprofielen en geklikte punten op te slaan.


# Technische informatie

De Kliktool is geschreven in de open source versie van Qt toolkit versie 5.9.1 en draait onder Windows, MacOS en Linux. Aangezien de Kliktool open source is ontwikkeld, is het mogelijk om de Kliktool verder te ontwikkelen of aan te passen en geschikt te maken voor gebruik in je eigen organisatie en processen. 


# Achtergrondinformatie

Deze Kliktool betreft de in augustus 2017 aangepaste Kliktool; qDAMEdit. De ‘oude Kliktool’ was wat verouderd en was aan een update toe. Deze is als basis gebruikt voor deze nieuwe versie en er is functionaliteit aan toegevoegd, zoals het op kunnen slaan van plaatjes, het toekennen van labels aan geklikte profielen en een logboekfunctie.
